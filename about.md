## Descripción y Unidades

Este curso profesional busca desarrollar las competencias lógico-matemáticas fundamentales para la computación. El contenido se divide en las siguientes unidades temáticas principales:

1. **Unidad 1:** Lógica Proposicional
2. **Unidad 2:** Lógica Cuantificacional
3. **Unidad 3:** Conjuntos y Relaciones.
4. **Unidad 4:** Algebra Booleana y sistemas numericos.
